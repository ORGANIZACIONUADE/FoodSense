# Swagger / OpenAPI

Este folder contiene el contrato inicial de API para FoodSense.

## Archivo principal

- `openapi.yaml`: especificacion OpenAPI 3.0.3 para US-04 a US-10.

## Como abrirlo rapido

1. Ir a [Swagger Editor](https://editor.swagger.io/)
2. `File` -> `Import file`
3. Seleccionar `Documentacion/swagger/openapi.yaml`

## Objetivo

Usar este contrato para coordinar frontend, backend y QA antes de implementar el backend real.
